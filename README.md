# TeavanaFrontEnd
To Run:
nodejs must be installed on machine
open command prompt (or mac equivalent) in the project root directory

          ex: c:/users/YOURPATH/info445/teavanafrontend/

type 'node server.js'
enter

will load the server to localhost:3000
